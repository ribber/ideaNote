# ideaNote

Use IDEA as markdown note software like Obsidian

feature
- [ ] support front-matter. when search a file，alias will be added to result list
- [x] move file to another directory，it's attachment will move too
- [x] input `[XXX]` auto complete note's relative path
- [x] extract selected text to a new note，and leave a link
- [x] find all unlink file
- [x] random open a note 


将 Idea 作为笔记软件，就像 Obsidian 那样


功能列表
- [ ] 支持 front-matter 中的同义词设置，搜索文件时支持搜索同义词
- [x] 移动笔记时，其附件也会随之移动
- [x] 添加笔记链接时，输入 `[XXX]` 弹出提示模糊搜索 xxx 笔记，点击回车自动补全笔记的相对路径
- [x] 将选择内容提取为一篇新的笔记，第一行作为文件名，并留下链接
- [x] 增加 inspection，检查所有没有被引用的附件，检查没有与其他笔记关联的笔记
- [x] 增加漫游笔记的按钮，随机打开一篇笔记
